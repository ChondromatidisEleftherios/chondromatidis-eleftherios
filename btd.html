Binary to Decimal Converter, is a Mips Assembly Desktop Application made in late 2023 as part of the "Computer Architecture" class. It's a simple program where the user types a Binary Number, and by pressing the "Enter" key, they can see the Decimal version of that number.

As you can see, while the decimal version appears, there's also a message confirming that the number inserted was indeed a number in the Binary System. If the user attempts to type a number that is not on the binary system, an error message will appear.
