NtomatoTimer is a Desktop Application created in 2023. It is a "Pomodoro-Timer" type of app, which aims in helping with concentration issues when doing an important task. How it works is very simple:

There's a "work" timer (that has the default value of 25 minutes), which displays the time the user has to do a task, and a "break" timer (that has the default value of 5 minutes), which shows the time the user has to rest.

The user also has the ability to set their own values for the timers by typing "2" on the cmd. (IT ONLY ALLOWS INTEGER VALUES)!

If something accurs and the user has to temporarily stop the timers, they can simply do that by pressing the "Space" key on their keyboard. If they wish to resume, they can do that by pressing any other key on their keyboard, APART from the "Space" key.

The user also has the ability to completely stop and exit a Timer session mid-way by pressing the "Escape" key and then "1" on their keyboard.

Lastly, the user can view their Completed Sessions and Total "Work" Time by pressing "3" while on the main menu. Those statistics are kept in a file, which ensures that even if the app is closed, the next time the user opens it, the statistics will still be there like they should be.
